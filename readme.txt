== Description ==

This research plugin allows you to search all of Amazon's Products by use of a Drill-down Category list search as well as by inputting specific item text to locate products to resell on your own pages (or elsewhere) by Country, Item Description, ASIN, Salesrank, Price and by Who Sells each product. You can also view every product as well for full details.

= Benefits =

Results show ASIN for each item as well as the rarely seen "Who sells" this item.

= Example usage =

Select A Category (and/or Sub Categories), enter a search item (or leave blank for All), select how many results to display and click Search.

== Installation ==

1. Upload the FOLDER 'Amazon Elite' to /wp-content/plugins/.
2. Activate the plugin 'Amazon Elite' through the 'Plugins' menu in admin.
3. Go to 'Settings' and then 'Amazon Elite' to Set Parameters.
